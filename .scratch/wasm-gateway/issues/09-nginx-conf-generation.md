# 09 — Resource Model → nginx.conf 생성

## What to build

파싱된 Resource Model(Listener/Router/Service/Policy)로부터 완성된 `nginx.conf`를 생성하는 `template.rs`를 구현한다. `gw-admin generate` 커맨드로 stdout에 출력하거나 파일로 저장할 수 있다.

매핑 규칙:
- Listener → `server { listen <port>; }`
- Service → `upstream <name> { server <host>:<port>; }`
- Router rules → `location <path> { proxy_pass; }`
- Policy → `spec.order` 오름차순 정렬 후 `proxy_wasm <filter> '<config_json>'` 나열
- Policy-Security(`apiKey`) → `api_key_filter`
- Policy-Traffic(`rateLimit`) → `rate_limit_filter`
- Policy-Transform(`headerControl`) → `header_manipulation_filter`

## Acceptance criteria

- [ ] `gw-admin generate` 실행 시 유효한 nginx.conf가 출력된다
- [ ] 생성된 nginx.conf를 `nginx -t`로 검증하면 통과한다
- [ ] Policy가 `spec.order` 기준으로 정렬되어 `proxy_wasm` 지시어가 나열된다
- [ ] `gateway/config/` 현재 리소스 파일로 생성한 nginx.conf로 `just integration`이 통과한다
- [ ] 생성된 `gateway/nginx/nginx.conf`는 gitignore에 추가된다

**Current behavior:**
nginx.conf가 수동으로 작성돼 있다. `gateway/config/` Resource Model 파일과 nginx.conf 사이에 자동 변환 로직이 없다.

**Desired behavior:**
`gw-admin generate`가 `GatewayConfig`를 읽어 nginx.conf를 stdout으로 출력한다. 생성된 파일은 `gateway/nginx/nginx.conf`에 저장되며 gitignore에 추가된다.

Resource Model → nginx.conf 매핑:
- `Listener` → `server { listen <port>; server_name <host>; }` 블록
- `Service` → `upstream <metadata.name> { server <host>:<port> weight=<weight>; keepalive 32; }` 블록  
- `Router.rules[].path` → `location <path> { proxy_pass http://<service>; }` 블록
- `Policy` → Router가 속한 location 블록 안에 `spec.order` 오름차순으로 `proxy_wasm <filter_name> '<config_json>'` 나열
- `PolicyConfig::Security.api_key` → `api_key_filter` + `{"api_keys":[...]}` JSON
- `PolicyConfig::Traffic.rate_limit` → `rate_limit_filter` + `{"requests_per_second":N}` JSON
- `PolicyConfig::Transform.header_control` → `header_manipulation_filter` + request/response add/remove JSON
- `logging_filter`는 항상 마지막에 추가 (order 999 고정)

**Key interfaces:**
- `fn render(config: &GatewayConfig) -> String` — nginx.conf 전체 문자열 반환
- `gw-admin generate [--out <path>]` — stdout 또는 파일로 출력
- `gateway/nginx/nginx.conf`를 gitignore에 추가
